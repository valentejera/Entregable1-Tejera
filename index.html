Simulador de Importación de Productos Venezolanos

Este simulador permite al usuario seleccionar productos venezolanos para importar dentro de Europa. Podrá elegir entre una lista de productos típicos (como Harina PAN, Chocolate Savoy, o Café Fama de América), calcular el costo total según el país de destino y confirmar su orden. Toda la interacción se hace desde la consola JS.
